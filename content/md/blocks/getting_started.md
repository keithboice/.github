### Install

```bash
% git clone git@github.com:keithboice/${{ env.REPOSITORY_SLUG }}.git

% cd ${{ env.REPOSITORY_SLUG }}

% npm install
```

### Develop

```bash
% npm run dev
```

### Test

```bash
% npm run test
```

### Flow

```bash
% npm run flow:coverage
```

### Documentation

```bash
% npm run docs

% npm run docs:html

% npm run docs:md
```

### Production

```bash
% npm run start
```
